
-- SUMMARY --

An Advanced tool to attach lazy load the images based on context.
Attaching context gives a huge flexibility. You can also set lazy load to a part of the page and with your choice of delay time.

It can immensely help if page performance.

For a full description of the module, visit the project page:
  http://drupal.org/project/context_image_loader
To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/context_image_loader


-- REQUIREMENTS --

Context

-- INSTALLATION --

Standard installation


-- CONFIGURATION --

Visit /admin/structure/context/context-lazy-load and have the basic settings.

-- USAGE --

1) Set up the configurations
3) Add new context, set the condition. In the reaction section, you would find options for image lazyloading.
4) Select DOM element wre lazy load is to be attached
5) You can add as many as you can with different load times.
